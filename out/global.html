easySongView(note, index, correctAnswers, noteClass, songName, limit)

easySongView generates the html for the notes of easy song for the introduction songs.
Parameters:
Name Type Description
note string The note that is being rendered to the view.
index integer The index coming from the inputted note array. Needed for the id of the img.
correctAnswers integer the number of correct answers in the lesson so far
noteClass string the ClassName that renders the note as a correct or incorrect note
songName string the name of the song. Needed for the id of the img.
limit integer the length of the inputted note array. Once it is reached, 3 blank images will render.
